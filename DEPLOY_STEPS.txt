=== Hunyuan3D-2.1 部署步骤 ===

[1] 安装 PyTorch Nightly (支持 RTX 5090 sm_120)
  uv pip install --upgrade torch torchvision torchaudio --index-url https://download.pytorch.org/whl/nightly/cu126 --python "G:\GitHub\Hunyuan3D-2.1\.venv\Scripts\python.exe"

[2] 构建 custom_rasterizer C++ 扩展（装好后重试）
  cd G:\GitHub\Hunyuan3D-2.1
  uv pip install -e hy3dpaint\custom_rasterizer --no-build-isolation --python ".venv\Scripts\python.exe"

[3] 编译 DifferentiableRenderer
  cd G:\GitHub\Hunyuan3D-2.1\hy3dpaint\DifferentiableRenderer
  ".venv\Scripts\python.exe" compile_mesh_painter.py

[4] 下载 RealESRGAN 权重
  curl -L -o G:\GitHub\Hunyuan3D-2.1\hy3dpaint\ckpt\RealESRGAN_x4plus.pth https://github.com/xinntao/Real-ESRGAN/releases/download/v0.1.0/RealESRGAN_x4plus.pth

[5] 启动 API Server
  cd G:\GitHub\Hunyuan3D-2.1
  .venv\Scripts\python.exe api_server.py --port 8081 --low_vram_mode
